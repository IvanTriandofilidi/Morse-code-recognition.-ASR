# Morse-code-recognition.-ASR
Модель машинного обучения для автоматического преобразования аудиофайлов, зашифрованных азбукой Морзе, в текст.
Система принимает на вход аудиофайл с сигналами азбуки Морзе и преобразует его в текстовую последовательность.
Для обработки временных и спектральных характеристик сигнала используется гибридная архитектура CNN + LSTM.

# Архитектура:
- Входные данные: аудиофайл (WAV)
- Предобработка: преобразование в спектрограмму
- Нейронные сети:
  - Convolutional Neural Networks (CNN)
  - Long Short-Term Memory (LSTM)
- Функция потерь: (CTC Loss)
- Метрика качества: расстояние Левенштейна

Расстояние Левенштейна - измеряет минимальное количество операций (вставка, удаление, замена),
необходимых для преобразования предсказанного текста в эталонный.

# Пример результата:
Вход: audio_01.wav
Выход: HELLO WORLD

# Используемые технологии:
